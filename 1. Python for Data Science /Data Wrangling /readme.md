# Data Wrangling

This project demonstrates essential Data Science and Machine Learning concepts using a dataset named employee-data.csv. The dataset contains employee details such as name, department, salary, experience, age, job role, and performance indicators. Using Python libraries like Pandas, NumPy, Matplotlib, and Scikit-Learn, this project performs complete data analysis and predictive modeling.
